# BOT_IRINA_PSYCHOLOGIST

Telegram bot for scheduled posts and admin management.

## Features
- Scheduled posting
- Admin panel
- Persistent tasks on restart

## Setup
1. Create `.env` from `.env.example`
2. Install dependencies:
```bash
   pip install -r requirements.txt
```
3. Run:
```bash
python bot.py
```

## Configuration

All config is stored in .env
